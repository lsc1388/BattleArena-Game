# 🎯 BattleArena Game

一個使用 Python 和 Pygame 開發的射擊對戰遊戲，玩家可以選擇不同角色與 AI 對手進行激烈的戰鬥！

## 🎮 遊戲特色

### 🦸‍♂️ 角色系統

- **特殊技能系統**：每個角色都有獨特的技能（30 秒冷卻時間，消耗 10%生命值）
  - 貓咪：激光攻擊 - 發射強力激光束
  - 狗狗：火焰攻擊 - 釋放火焰傷害
  - 狼：冰凍攻擊 - 發射冰凍彈

### 🔫 武器與戰鬥

- **多武器系統**：手槍、步槍、散彈槍可自由切換

### 💎 道具系統

- 🔥 **火力增強**：提升子彈攻擊力
- 🎯 **彈藥補給**：補充所有武器彈藥
- 💥 **散彈模式**：單次發射五顆子彈

### ⚙️ 自訂設定

- **生命值設定**：可調整初始血量（50-200）
- **血量顯示**：支援血條和數字兩種顯示模式
- **難度選擇**：三種 AI 智能等級

## 🕹️ 操作控制

### 角色移動與戰鬥

- `W/A/S/D` - 角色移動
- `滑鼠左鍵` - 射擊
- `滑鼠右鍵` - 重新開始遊戲
- `R` - 手動填裝
- `Q` - 使用角色特殊技能

### 武器切換

- `1` - 切換至手槍
- `2` - 切換至步槍
- `3` - 切換至散彈槍

### 系統操作

- `ESC` - 暫停/返回選單
- `H` - 切換血量顯示模式（選單中）
- `+/-` - 調整初始血量（選單中）
- `空白鍵` - 確認選擇（選單中）

## 🚀 快速開始

### 系統需求

- Python 3.7 或更高版本
- Pygame 2.0 或更高版本

### 安裝步驟

1. **克隆專案**

   ```bash
   git clone <repository-url>
   cd BattleArena-Game
   ```

2. **安裝依賴**

   ```bash
   pip install pygame
   ```

3. **運行遊戲**
   ```bash
   python main.py
   ```

## 🎯 遊戲玩法

### 開始遊戲

1. 運行遊戲後進入主選單
2. 調整遊戲設定（難度、血量等）
3. 按空白鍵進入角色選擇
4. 選擇你喜歡的角色（貓咪/狗狗/狼）
5. 選擇戰鬥場景（熔岩/高山/冰雪）
6. 開始戰鬥！

### 戰鬥策略

- **角色技能運用**：善用每個角色的獨特特殊技能
- **武器選擇**：合理運用三種武器的特性
- **道具收集**：撿取道具增強戰力
- **彈藥管理**：注意彈藥量，適時填裝
- **敵人類型識別**：不同敵人有不同能力，需要不同應對策略

### 得分系統

- 擊敗敵人：+100 分
- 拾取道具：+50 分
- 追求更高分數和生存時間！

## 🔧 自訂與擴展

### 修改遊戲設定

編輯 `src/config.py` 檔案可調整：

- **角色設定**：角色技能效果、冷卻時間、生命值消耗
- **場景設定**：場景背景顏色、視覺效果
- **武器屬性**：傷害、射速、彈藥容量
- **AI 行為參數**：精確度、移動速度、敵人類型能力
- **道具效果**：道具效果強度和持續時間
- **畫面設定**：尺寸和幀率

### 添加新功能

遊戲採用模組化設計，可輕鬆擴展：

- **新增角色類型**：在 CHARACTER_CONFIGS 中添加新角色
- **新增場景環境**：在 SCENE_CONFIGS 中添加新場景
- **設計新的 AI 類型**：在 AI_ENEMY_TYPES 中添加新敵人
- **創建更多道具效果**：擴展 PowerUp 系統
- **實作多人對戰模式**：基於現有架構添加多人支援
